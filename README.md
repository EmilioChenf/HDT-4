# HT4 - Calculadora Infix/Postfix

Implementación de una calculadora que convierte expresiones infix a postfix y las evalúa.

  - Conversión Infix a Postfix
  - Evaluación de expresiones
  - FileReader

- **Arturo**: Listas Enlazadas
